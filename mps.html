MPS - mikroprocesorski sistemi

Tipi spremenljivk

  • Celoštevilske spremenljivke:

    • Byte (8b = 1B) -> 2^8 [0...255]

    • Int (16b = 2B) -> 2^16 [-2^15...0...+2^15-1]

    • Long (32b = 4B) -> 2^32 [-2^31...0...+2^31-1]

    • Unsigned Int/Long => Samo pozitivna števila

  • Decimalne spremenljivke:

    • Float (32b = 4B) => 2^32

    • Double (64b = 8B) => 2^64

  • Logične spremenljivke:

    • Bool (8b = 1B) => True/False (False = 0, True != 0)

  • Znaki:

    • Char (8b = 1B) => 'a','b',... => ASCII tabela

Array/Nizi

  • Definiranje arraya:

    • char CHAR_ARR[20]; => definiramo prazen array, ki ima možnost sprejeti do 20 elementov tipa char.

    • int INT_ARR[20] = {0,1,2}; => naredimo array integerjev, v katerega lahko gre do 20 elementov tipa integer

    • int LOTO = {1,2,3,4,5,6}; => naredimo array integerjev, ki ima 6 prostorov(že poln array)
